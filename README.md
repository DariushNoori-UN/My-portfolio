# Portfolio: Dariush Noori

Welcome to my personal portfolio project. This repository contains a small static website built in HTML, CSS, and optional JavaScript showcasing my profile, skills, and projects.

## 📁 Contents

- `index.html` – Main landing page
- `about.html` – About me and experience
- `style.css` – Shared styling for both pages
- `Images/` – Asset folder (logo and profile pictures)

## 🚀 Features

- Responsive navigation and hero section
- Career highlights and interests
- Social links (GitHub, LinkedIn, Facebook, Instagram)
- Easy to extend with new sections or projects
